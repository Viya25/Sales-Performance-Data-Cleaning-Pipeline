Sales Performance & Data Cleaning Pipeline
Files: sales_data_cleaned.xlsx, Sales Dashboard.xlsx, Problem Statement.pdf

Project Overview: A focused project on the "Data Wrangling" phase of analytics. Starting from a complex problem statement, this project involves transforming raw, messy sales records into a clean, analysis-ready dataset and a final performance dashboard.

Key Features:

Problem Solving: Interpreted business requirements from a formal Problem Statement to define project scope.

Data Wrangling: Performed extensive cleaning (handling duplicates, null values, and formatting errors) to produce the sales_data_cleaned file.

KPI Tracking: Developed a secondary Sales Dashboard to visualize the impact of the cleaned data on business metrics like total revenue and regional growth.

Reporting: Documented the transformation process and final results for stakeholder review.
